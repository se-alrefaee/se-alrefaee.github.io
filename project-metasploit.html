Windows Exploitation Lab

Metasploit Penetration Testing & Meterpreter Control

Comprehensive penetration testing exercise using Metasploit framework to exploit Windows systems, establish reverse shells, and demonstrate post-exploitation techniques including keylogging and screen capture.

Project Overview

This lab simulates a controlled penetration test against Windows systems. Using Metasploit, I demonstrated realistic attack scenarios including payload delivery via phishing, establishing persistent access, and post-exploitation activities. The objective was to understand both offensive and defensive perspectives for building robust security architectures.

Framework
Metasploit Pro
Target OS
Windows Server 2022
Attack Type
Reverse Shell / Meterpreter

Attack Flow

1

Lab Setup

Configured Kali Linux and Windows Server 2022 virtual machines. Set up isolated network environment for safe testing and controlled execution.

2

Payload Creation

Generated custom Metasploit payloads. Configured reverse shell connections and listener setup for command execution.

3

Delivery

Simulated phishing attack vector. Executed payload delivery through social engineering to establish initial access.

4

Post-Exploitation

Leveraged Meterpreter sessions for advanced reconnaissance. Executed keylogging, screen capture, and credential harvesting techniques.

Key Techniques & Learning

Payload Generation

Mastered Metasploit payload creation for various encoding and obfuscation techniques. Bypassing antivirus detection and execution policies.

Meterpreter Sessions

Comprehensive understanding of Meterpreter capabilities. System command execution, file manipulation, and process enumeration.

Keylogging

Implemented keyloggers to capture user input and credential harvesting. Understanding persistence and covert data exfiltration.

Screen Capture

Utilized screenshot functionality for visual reconnaissance. Monitoring user activities and sensitive information extraction.

Privilege Escalation

Learned escalation vectors and exploitation of Windows privilege models. UAC bypass techniques and local exploit execution.

Defensive Insights

Understanding attack methods informs better defensive strategies. Identifying detection evasion techniques to build better protections.
