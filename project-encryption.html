Encryption Labs

Symmetric & Asymmetric Encryption Fundamentals

Hands-on cryptography lab exercises exploring MD5, SHA256 hashing and symmetric/asymmetric encryption. Practical file encryption and decryption with peer collaboration to understand cryptographic principles.

Lab Overview

Comprehensive cryptography lab covering fundamental encryption concepts and practical implementations. Explored both symmetric encryption (shared keys) and asymmetric encryption (public/private key pairs). Collaborated with peers to encrypt, transmit, and decrypt sensitive information securely.

Focus
Cryptographic Principles
Algorithms
MD5, SHA256, RSA, AES
Method
Practical File Encryption

Encryption Concepts Explored

1

Hashing Fundamentals

Learned MD5 and SHA256 hashing algorithms. Understood one-way functions and their role in data integrity verification.

2

Symmetric Encryption

Practiced AES encryption with shared keys. Encrypted and decrypted files using symmetric algorithms for fast data protection.

3

Asymmetric Encryption

Explored RSA key pairs and public/private key cryptography. Implemented secure key distribution and encryption without pre-shared secrets.

4

Peer Collaboration

Exchanged encrypted messages with colleagues. Each party encrypted with recipient's public key, demonstrating secure communication protocols.

Key Learning Outcomes

Encryption Techniques

Comprehensive understanding of both symmetric and asymmetric encryption methods. When to apply each technique based on security requirements and performance constraints.

Key Management

Best practices for cryptographic key generation, storage, and rotation. Understanding key derivation functions and secure key exchange protocols.

Data Protection

Practical implementation of encryption for file protection and secure data transmission. Understanding confidentiality and integrity verification.
